# 包管理工具

## 1. 前言（why，what）

  项目往往由多个软件包构成，如果没有软件包管理工具，每位开发者不得不手动完成以下工作：

  1. 记录项目中依赖的所有软件包；
  2. 手动下载、安装、更新、删除所有软件包；
  3. 跟踪软件包的安全问题、最新更新版本；

  包管理工具通过在项目中配置文件，协助开发者自动完成上述工作。下面介绍前端项目中常用的两种包管理工具：**NPM**（Node Package Manager） 和 **YARN**（Yet Another Resource Negotiator）。
